## 📚 What's Inside? (14 Days of Fun)

Here are the concepts covered with fun, real-world analogies:

1. **Day 1: Magic Boxes** - Variables (`let`, `const`, `var`)
2. **Day 2: Age Calculator** - Data Types & Operators
3. **Day 3: Making Choices!** - Conditions (`if`, `else`, `switch`)
4. **Day 4: Jumping Loops!** - Loops (`for`, `while`, `do while`)
5. **Day 5: Helper Robots** - Functions
6. **Day 6: Toy Train** - Arrays (`push`, `pop`)
7. **Day 7: Superhero ID Card** - Objects & Methods
8. **Day 8: Magic Spells** - Array Methods (`map`, `filter`)
9. **Day 9: Magic Wand** - DOM Manipulation
10. **Day 10: Action Events!** - Events (`click`, `input`, `submit`)
11. **Day 11: Modern Shortcuts** - ES6 Features (Arrow Functions, Destructuring)
12. **Day 12: Fetch (Talking to Internet)** - Async JS (`Promises`, `async/await`, Fetch API)
13. **Day 13: Secret Backpack** - Storage (`localStorage`, `sessionStorage`)
14. **Day 14: My Fun To-Do List** - Mini Project putting it all together!

## 🛠️ How to run locally?

1. Clone this repository.
2. Open `index.html` in your favorite web browser.
3. Click the buttons, swap toys, send pigeons, and have fun! 🚀
